### Installation

Requirements: `tmux` version 1.9 (or higher), `git`, `bash`.

Clone TPM:

```bash
# Debian/Ubuntu
sudo apt install subversion

# macOS (Homebrew)
brew install subversion
```
Next :
```bash
git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
```
```bash
svn export
```
